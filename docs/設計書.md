# 画面遷移
画面遷移を次の図で示します。

```mermaid
graph TD
    top["TOP画面"]--簡単無料見積もりをする-->input["入力画面"]
    input--TOPへ戻る-->top
    input--ご入力内容の確認へ-->confirm["確認画面"]
    confirm--入力画面へ戻る-->input
    confirm--概算お見積りをする-->result["見積もり結果画面"]
    result--TOPへ戻る-->confirm
    result--詳細見積もり依頼へ-->complete["完了画面"]
    complete--TOPへ戻る-->top
```

※ 各画面、ヘッダーの会社ロゴを押下することでTOP画面に遷移する。
